# Pricing Record Management

This project is a Spring Boot application for managing pricing records. It provides RESTful APIs to upload, search, and update pricing records.

## Technologies Used

- Java 11
- Spring Boot 2.3.4
- Spring Data JPA
- MySQL
- Gradle
